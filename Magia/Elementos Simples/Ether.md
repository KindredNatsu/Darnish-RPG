# Darnish RPG: Elementos Simples // Ether

O elemento usado para explicar o dívino, muitos conhecem a existência do Ether, mas poucos consegue manipularem, normalmente se limitando a mestres anciões e sarcedotes, pois apenas aqueles que tem ligação com o mítico e são dignos disso, podem e conseguem usar do Ether ao seu favor

### Efeitos Práticos

- Suporte seja com curas ou com aumento de força
- Aumentar suas capacidades mágicas
